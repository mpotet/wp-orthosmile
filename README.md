# OrthoSmile — Thème WordPress pour Cabinets d'Orthodontie

Thème WordPress premium conçu pour les cabinets d'orthodontie et cliniques dentaires. Installez-le en quelques minutes et personnalisez tout depuis le panneau d'administration.

---

## Installation rapide

1. Téléchargez le fichier `.zip` du thème OrthoSmile.
2. Dans votre tableau de bord WordPress, allez dans **Apparence → Thèmes → Ajouter**.
3. Cliquez sur **Téléverser un thème**, sélectionnez le `.zip`, puis cliquez sur **Installer maintenant**.
4. Cliquez sur **Activer**.

---

## Personnalisation via le panneau d'administration

Allez dans **Apparence → Personnaliser** pour modifier le thème sans toucher au code.

### 📞 Coordonnées
| Champ | Description |
|---|---|
| Numéro de téléphone | Affiché dans l'en-tête et le pied de page |
| Email | Email de contact |
| Adresse | Adresse du cabinet |
| Horaires d'ouverture | Plages horaires affichées |
| URL rendez-vous | Lien vers votre outil de réservation en ligne |
| Google Maps | Code d'intégration pour afficher votre carte |

### 🎨 Couleurs
Choisissez librement la **couleur primaire** (défaut : vert teal) et la **couleur d'accent** (défaut : doré) pour correspondre à votre identité visuelle.

### 🖼️ Section Héro (bannière d'accueil)
- Titre et sous-titre personnalisables
- Image de fond au choix
- Texte et URL du bouton d'appel à l'action

### 📊 Section Confiance
- Personnalisez les 3 statistiques clés (ex. : années d'expérience, nombre de patients)

### 👨‍⚕️ Spécialistes
- Activez ou désactivez cette section
- Noms, titres et descriptions des 3 praticiens

### 💬 Témoignages
- Activez ou désactivez la section
- Personnalisez les textes et auteurs

### 📋 Sections
- Activez/désactivez chaque section depuis **Affichage des Sections** : Services, Galerie, Équipe, Témoignages, FAQ, Appel à l'action.

### 🔗 Réseaux sociaux
- Ajoutez vos liens Facebook, Twitter, Instagram et LinkedIn. Les icônes apparaissent automatiquement.

### 🖼️ Logo
Allez dans **Apparence → Personnaliser → Identité du site** pour uploader votre logo.

---

## Menus de navigation

1. Allez dans **Apparence → Menus**.
2. Créez votre menu et assignez-le à l'emplacement **Menu principal** ou **Menu footer**.

---

## Formulaire de contact

Le formulaire de contact est intégré. Les messages sont envoyés à l'adresse email de votre administrateur WordPress. Aucun plugin supplémentaire n'est requis.

---

## Traduction et multilingue

Le thème est entièrement traduisible. Il est compatible avec les plugins de traduction populaires :
- **Polylang**
- **WPML**
- **TranslatePress**

Le fichier de base de traduction (`orthosmile.pot`) se trouve dans le dossier `languages/`. Vous pouvez l'utiliser pour créer une traduction avec [Poedit](https://poedit.net/).
